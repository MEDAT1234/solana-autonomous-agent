\# 🤖 Solana Autonomous Agent – MCP Server with 420 Protocol \& Real Swaps



This is a complete \*\*Model Context Protocol (MCP) server\*\* that acts as an autonomous agent on Solana Devnet. It can:



\- 💰 Manage a Solana wallet (auto-created or imported)

\- 🔁 Request SOL from faucet with automatic retry

\- 🔄 Execute real token swaps via Jupiter API (SOL → USDC)

\- 💸 Simulate \*\*420 protocol payments\*\*

\- 🔧 Discover tools using \*\*Synapse Agent Protocol (SAP)\*\*
